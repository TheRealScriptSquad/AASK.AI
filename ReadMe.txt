AI Analysis Of Student Knowledge ( AASK.AI )
============================================

Description:
-------------
This project aims to make an efficient computerized mechanism for predicting student grades using Artificial Intelligence. It accepts the educational and socio-economic conditions of a student and attempts to predict the grades. It is made using Python with the help of various libraries and can be deployed on a local host. It is also equipped with a user friendly UI and can effectively predict student grades with upto 90% accuracy.

Prerequisites:
-------------
   - python 3.x version

Usage:
-------------
1. Configuration:
   - pip install -r requirements.txt

2. Running the Project:
   - open the project folder in terminal.
   - run the flask server by the command - ` python app.py `
   - this will generate a url that will direct to the predictor webpage
